概述

window.cancelIdleCallback() 方法用于取消之前调用{{domxref("window.requestIdleCallback()")}} 的回调。

语法

window.cancelIdleCallback(handle);

参数

handle
调用 {{domxref("window.requestIdleCallback()")}} 时返回的 ID.

返回值

undefined.
