{{APIRef("DOM Events")}}

WheelEvent.deltaX 只读属性是一个 double 类型值，声明水平滚动量以{{domxref("WheelEvent.deltaMode")}} 为单位。

语法

var dX = event.deltaX;

例子

var syntheticEvent = new WheelEvent("syntheticWheel", {"deltaX": 4, "deltaMode": 0});

console.log(syntheticEvent.deltaX);
