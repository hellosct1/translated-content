关键字 prefetch 作为元素 {{HTMLElement("link")}} 的属性 {{HTMLAttrxRef("rel", "link")}} 的值，是为了提示浏览器，用户未来的浏览有可能需要加载目标资源，所以浏览器有可能通过事先获取和缓存对应资源，优化用户体验。
